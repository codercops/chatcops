## Packages

| Package | Description |
|---------|-------------|
| `@chatcops/widget` | Embeddable chat widget (IIFE + ESM), Shadow DOM isolated, ~7KB gzipped |
| `@chatcops/server` | Server handler + Express, Vercel, Cloudflare adapters |
| `@chatcops/core` | AI provider abstraction, tools, knowledge base, i18n, analytics |

## Architecture

```
┌──────────────────┐         ┌──────────────────┐
│   Your Website   │  SSE    │   Your Server    │
│                  │ ──────► │                  │
│  <script> tag    │ ◄────── │  @chatcops/server│
│  @chatcops/widget│ stream  │  ├─ Provider     │
│  Shadow DOM      │         │  ├─ Knowledge    │
│  Chat Panel      │         │  ├─ Tools        │
└──────────────────┘         │  └─ Analytics    │
                             └──────────────────┘
```
